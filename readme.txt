== Description ==

"WPCF7 Extended" can add confirm step to "Contact Form 7".
WPCF7拡張はお問い合わせフォーム7を追加できます確認, に確認画面を追加するプラグインです。

WPCF7 Extended" can add confirm step and many other utilities :
* WPCF7 Extended Contact Form 7 Confirm : go to confirmation step.
* WPCF7 Extended Contact Form 7 Button : add option submit button as <button> instead of <input type="submit">.
* WPCF7 Extended Contact Form 7 Thanks : after sending the mail successfully, it will redirect to the installed path.

WPCF7 Extended "は、確認ステップおよび他の多くのユーティリティを追加できます。
* WPCF7拡張お問い合わせフォーム7確認：確認手順に進みます。
* WPCF7拡張お問い合わせフォーム7ボタン：オプション送信ボタンを<input type="submit">ではなく<button>として追加します。
* WPCF7拡張お問い合わせフォーム7ありがとうございます：メールを正常に送信した後、インストールされたパスにリダイレクトされます。

Plugin WPCF7 Extended is not related to the developer of "Contact Form 7".
プラグインWPCF7Extendedは、「ContactForm7」の開発者とは関係ありません。

== Installation ==

1. If you have not yet, Install Contact Form 7 Plugin
2. Upload the entire 'wpcf7-extended' folder to the '/wp-content/plugins/' directory.
3. Activate the plugin name 'WPCF7 Extended' through the 'Plugins' menu in WordPress.

* PHP version 5.4 or greater (PHP 5.6 or greater is recommended)
* MySQL version 5.5 or greater (MySQL 5.6 or greater is recommended)
